### Production Deployment 
1. **Environment Setup**: Configure production `.env` with real API keys
2. **Database Migration**: Run Flyway migrations against production database
3. **SSL Certificates**: Ensure HTTPS for voice webhook endpoints
4. **Load Balancing**: Configure load balancer for high availability
5. **Monitoring**: Set up Prometheus/Grafana for system monitoring


### Docker Production Build
```bash
# Build production image
docker build -t xlbiz-incident-agent:latest .

# Run with production configuration
docker run -d \
  --name incident-agent \
  --env-file .env.production \
  -p 8080:8080 \
  xlbiz-incident-agent:latest
```

### Kubernetes Deployment
```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: incident-agent
spec:
  replicas: 3
  selector:
    matchLabels:
      app: incident-agent
  template:
    metadata:
      labels:
        app: incident-agent
    spec:
      containers:
      - name: incident-agent
        image: xlbiz-incident-agent:latest
        ports:
        - containerPort: 8080
        envFrom:
        - secretRef:
            name: incident-agent-secrets
```
